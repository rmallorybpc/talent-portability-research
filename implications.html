Talent Portability Research

Implications

Guidance for compensation design, talent acquisition, and work structure.

Who to hire and when

Start with the value-creating asset in the role. If value depends on firm-specific resources, networks, and culture, it does not transfer cleanly. If value depends on portable assets like client relationships or portable technical skill, transfer is more likely.

Bidwell supports internal promotion for firm-specific roles. Berry-Stolzle and Gurun support external hiring where relationship books are portable. Roll and Boivie imply decision-makers should calibrate confidence downward, not upward, with experience.

How to make external hires successful

Plan for a two-year ramp

Bidwell's findings suggest external hires should be evaluated as long-term investments, not full-output contributors from day one.

Build firm-specific scaffolding

Groysberg's framework emphasizes that onboarding, internal networks, and process integration are part of the value transfer process.

Manage team moves in stages

The Groysberg and Abrahams lift-out framework gives four stages: courtship, leader integration with top leadership, work-process integration, then cultural integration.

Run a structured alternative check

For each premium external hire, compare against internal promotion, delayed hiring, and no-hire options before final pricing.

How to structure work for portability

Portability is also a design choice. If a firm wants value embedded in the organization, it should design systems that compound firm-specific output. If it wants portable books and relationships, it should design for that and accept governance trade-offs.

Gurun shows that unlocked client portability can increase misconduct and fees in some contexts, which means portability upside can come with governance costs.

Soft close

The synthesis points to a narrow set of conditions where paying for outside talent works reliably. If the pattern matches your organization, the next step is deciding which lever to pull first: hire strategy, compensation design, or work design.
